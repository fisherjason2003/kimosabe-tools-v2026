## Overview

Kimosabe Tools is a web-based utility collection built around common PDF and image tasks. Because processing happens on the client side, your files stay in the browser instead of being sent out to a remote service.

It is aimed at users who want fast document and media edits without installing dedicated desktop software. By combining PDF tools and image utilities in one interface, it supports simple, privacy-conscious workflows in modern browsers.

## Usage

Open the application in your browser and choose the tool that fits the job.

1. Load a PDF or image from your device.
2. Select an operation such as merge, compress, rotate, split, resize, or convert.
3. Tweak any settings that are available.
4. Preview the output if the tool supports it.
5. Save the finished file back to your device.

For PDF signing, follow the prompts in the app to apply your signature before exporting the document. For image tasks, choose the compression or conversion option you need, then download the processed file once it is ready.
